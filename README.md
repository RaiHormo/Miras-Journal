# Miras-Journal
A turn-based JRPG made with godot that is currently in early development.

## Trying the game
You may try one of the available releases on the right of the github interface. These are meant for playtesting and don't fully represent the full game. 

## Is this open source
In terms of code, Yes! The art assets are under copyright.
The game will still be paid when releasing on steam, but the source code will remain available.

## Documentation
The documentation includes things like story details, worldbuilding as well as technical info. These documents however are private, please ask me (Iris) for access if you would like to contribute.
